# relora

Install Relora from npm:

```bash
npm install -g relora
```

Then launch it:

```bash
relora
```

This package downloads a prebuilt Relora bundle for the current platform from GitHub Releases during `postinstall`.

The bundle includes:

- `relora`
- `relora-driver-postgres`
- `relora-driver-mysql`
- `relora-driver-sqlite`
